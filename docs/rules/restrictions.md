# スコープガード

※ CLAUDE.md の「実装の鉄則」と重複する内容は CLAUDE.md 側を正とする。ここでは補足的な詳細を記載。

## スコープ

- 実装範囲はタスクドキュメント（`docs/task/XX-*.md`）と要件定義（`docs/lunchy-requirements.md`）に記載された内容のみ
- リファクタリングはタスクとして明示的に指示された場合のみ行う

## 確認が必要な操作

- パッケージの追加・削除
- `any` 型の使用
- ドキュメントに記載のない判断

いずれも AskUserQuestion でユーザーに確認してから行う。

## 課題の記録

実装中に課題・疑問が出た場合は、該当ドキュメントの Implementation Q&A セクションに記録する。
