# Lunchy MVP タスク一覧

## 全体フロー

```
[01 Figma MCP セットアップ] → [02 UIデザイン制作（Figma）] ─┐
                                                              ├→ [07 セッション作成画面]
[03 プロジェクト初期構築]                                     │   [08 メンバー参加画面]
    ├→ [04 静的解析ルール設定]                                │   [09 好み入力フォーム]
    └→ [05 データモデル・API設計] ─────────────────────────────┘   [10 幹事ダッシュボード]
         │                                                        [11 投票画面]
         └→ [06 AI候補生成（Gemini + Google Maps）]               [12 結果表示画面]
                                                              → [13 レスポンシブ・UI仕上げ]
                                                              → [14 Vercelデプロイ]

※ 07-12 は「05 + 02」の両方が完了してから着手
※ 06 はUIに依存しないため 05 完了後に着手可能
※ 01→02 と 03→05 は並行して進行可能
```
