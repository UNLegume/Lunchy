# 11: 投票画面

## 概要

AI が提示した候補3件に対して、全メンバーが無記名で投票する画面。

## 画面仕様

### 第1回投票

- 候補3件をカード形式で表示
  - 店名・ジャンル・徒歩時間・評価・価格帯・写真1枚
- 1件を選択して「投票する」ボタン
- 投票後: 「投票完了！結果発表を待っています...」の待機画面

### 結果発表（第1回）

- 全員投票後に一斉公開
- 各候補の得票数を表示
- 過半数あり → 決定画面（タスク12）へ
- 過半数なし → 決選投票へ

### 決選投票

- 上位2件をカード形式で表示
- 1件を選択して「投票する」ボタン
- 同数の場合 → ランダムで決定
