# 12: 結果表示画面

## 概要

投票で決定した店を全メンバーに表示する画面。

## 画面仕様

### 決定画面

- 決定した店の情報を大きく表示
  - 店名（大きく）
  - ジャンル
  - 徒歩時間
  - 評価（星）
  - 価格帯
  - 写真
- 投票結果の内訳（各候補の得票数）
- Google Maps へのリンク（経路案内）
- 「お疲れさまでした！」的なメッセージ
