# 10: 幹事ダッシュボード

## 概要

幹事がセッションの進行を管理する画面。入力進捗の確認と締め切り操作を行う。

## 画面仕様

### 好み入力フェーズ

- 進捗表示: 「N / M 人回答済み」（リアルタイム更新）
- 「締め切る」ボタン（好み入力を締め切ってAI候補生成へ進む）
- ※ 誰が回答したかは非表示（無記名維持）

### 投票フェーズ

- 候補3件の表示
- 進捗表示: 「N / M 人投票済み」（リアルタイム更新）
- 全員投票後、結果が一斉公開される

### 決選投票フェーズ（必要な場合）

- 上位2件の表示
- 進捗表示: 「N / M 人投票済み」
- 全員投票後、結果が一斉公開される

## リアルタイム更新の実装

- 方式: ポーリング（3秒間隔）or Server-Sent Events
- MVP ではポーリングで十分
