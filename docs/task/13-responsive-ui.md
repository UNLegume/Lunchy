## 概要

全画面をPC・スマホ両方で快適に使えるようにUIを仕上げる。

## やること

- [ ] 全画面のスマホ表示確認・調整
- [ ] 全画面のPC表示確認・調整
- [ ] ブレークポイント設定（Tailwind CSS）

- [ ] フォントサイズ・余白の調整
