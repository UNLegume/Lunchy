# 07: セッション作成画面

## 概要

幹事がセッションを作成し、共有URLを取得する画面。

## 画面仕様

### トップページ (`/`)

- サービス名「Lunchy」のロゴ・タイトル
- 「ランチセッションを作る」ボタン

### セッション作成フォーム

- 幹事の表示名入力
- オフィスの場所入力（住所 or 地名）
- 「セッションを作成」ボタン

### 作成完了画面

- 発行されたセッションURL表示
- 「URLをコピー」ボタン
- 「セッションに進む」ボタン（幹事自身がダッシュボードへ）

## やること

- [ ] トップページ UI 作成
- [ ] セッション作成フォーム UI 作成
- [ ] `POST /api/sessions` の実装
- [ ] URL発行ロジック（UUID生成）
- [ ] URLコピーボタンの実装
- [ ] 作成完了後、幹事のダッシュボードへの導線

## 完了条件

- 幹事がフォームに入力してセッションを作成できる
- セッションURLが発行されコピーできる
- KV にセッションデータが保存される
